# LeetCode Lookalike Full-Stack Project

This is a full-stack application mimicking LeetCode, with a Next.js frontend and a Node.js backend.

## Structure

- `frontend/`: Next.js application with ProblemList component
- `backend/`: Node.js Express server with problems API

## Features

- Display list of problems with title, category, difficulty
- Backend API to serve problems data
- Styled with Tailwind CSS

## Getting Started

### Frontend
1. Navigate to `frontend/`
2. Run `npm install`
3. Run `npm run dev`

### Backend
1. Navigate to `backend/`
2. Run `npm install`
3. Run `npm start`

The frontend will proxy API calls to the backend.
